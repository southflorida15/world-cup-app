# Changelog

All notable changes to this project will be documented in this file.

## [3.1.0] - 2026-06-26

### Added

- New `Insights` tab for Match OG.
- New `MATCH FLOW` naming for the match momentum visualization.
- Dedicated Commentary tab.
- Separate yellow-card and red-card indicators in the event pills.
- Momentum engine module at `engine/momentum/momentumEngine.js`.
- Match-event cache flush maintenance action.

### Changed

- Renamed `Match Stats` tab to `Stats`.
- Reorganized Match OG tabs for clearer match navigation.
- Refined Match Flow wording and supporting insight labels.
- Improved match event and card display logic.
- Improved final Match Flow visual tuning.

### Fixed

- Commentary tab black-screen issue caused by an undefined component.
- Red cards being counted under the yellow-card-only pill display.
- Several Match OG layout and rendering edge cases.
