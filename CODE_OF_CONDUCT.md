# Code of Conduct

## Our Pledge

We want World Cup App to be a respectful, welcoming, and constructive project.

Everyone participating in this project is expected to treat others with respect.

---

## Expected Behavior

Examples of positive behavior include:

- Being respectful and constructive.
- Welcoming different perspectives.
- Giving helpful feedback.
- Focusing on the project and the work.
- Assuming good intent.
- Being patient with questions and contributions.

---

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment or intimidation.
- Discriminatory comments.
- Personal attacks.
- Trolling or insulting language.
- Publishing private information without permission.
- Disruptive conduct that harms the project community.

---

## Scope

This Code of Conduct applies to:

- Issues.
- Pull requests.
- Discussions.
- Documentation.
- Project-related communication.

---

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or pull requests that do not align with this Code of Conduct.

Repeated or serious violations may result in being blocked from participating in the project.

---

## Goal

The goal is simple:

Build a great project and treat people well while doing it.
