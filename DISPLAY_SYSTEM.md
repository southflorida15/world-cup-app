# World Cup App – Display System

## Minute Display (Canonical)

### First Half
- `12'`
- `38'`

### First-half stoppage
- EN: `45'+3' (45'+3' H1)`
- PT: `45'+3' (45'+3' 1T)`

### Second Half
- EN: `67' (22' H2)`
- EN: `90'+5' (45'+5' H2)`
- PT: `67' (22' 2T)`
- PT: `90'+5' (45'+5' 2T)`

### Extra Time
- EN: `93' (3' ET)`
- EN: `120'+5' (30'+5' ET)`
- PT: `93' (3' Pror.)`
- PT: `120'+5' (30'+5' Pror.)`

## Goal Labels
- Goal
- Penalty: `(PEN)` / `(PÊN.)`
- Own Goal: `(OG)` / `(contra)`

## Styling
- Official minute: bold, gold.
- Context: same line, ~90% font size, italic, muted color.
